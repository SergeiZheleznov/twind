# Interface: CSSRules

twind.CSSRules

See: https://drafts.csswg.org/css-nesting/#nest-selector

"& > * + *": {
  marginLeft: 16
},

// In a comma-separated list, each individual selector shall start with "&"
"&:focus, &:active": {
  outline: "solid"
},

// Self-references are also supported
"& + &": {
  color: "green"
}
